# Tetris Game #

  Tetris clone built in Python using Pygame

# Requirements #

  - Python 3.x
  - Pygame

  ### Controls ###
    left - left arrow key or a
    right - right arrow key or d
    down - down arrow key or s
    rotate - up arrow key or w

    pause/play - p key
    mute sound - m key

    replay button - enter key or mouse click on button area
